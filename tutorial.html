How to Use PyClef

A step-by-step guide to get started with PyClef for optical music recognition.

1. Installation

Install PyClef using pip:

$ pip install pyclef

2. Import the Library

Import PyClef in your Python script:

import pyclef

3. Load Your Sheet Music

Load an image of sheet music:

omr = pyclef.OMR()
image = omr.load_image('path/to/sheet_music.png')

4. Process the Image

Run the OMR pipeline:

result = omr.process(image)
print(result)

5. Export Results

Save the recognized music data:

omr.export(result, 'output.mid')
