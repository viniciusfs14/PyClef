## 🇧🇷 Português

### 📝 Sobre o Projeto
O **PyClef** é um pipeline de OMR (*Optical Music Recognition*) de alta performance. O projeto utiliza visão computacional de ponta para detectar símbolos musicais e princípios de Processamento de Sinais Digitais (DSP) para converter coordenadas espaciais em frequências Hz discretas.

Diferente de ferramentas tradicionais, o PyClef foca na **inferência bruta** e na precisão submilimétrica, permitindo a digitalização de partituras complexas, manuscritas ou impressas.

### 🚀 Tecnologias
- **Deep Learning:** Modelo **YOLO11m** ajustado para detecção de glifos musicais.
- **Interface Gráfica:** Desenvolvida em **PySide6 (Qt)** para uma experiência moderna.
- **DSP Engine:** Mapeamento matemático de notas em arquivos MIDI e áudio sintetizado.

## 🇺🇸 English

### 📝 About the Project
**PyClef** is a high-performance OMR (*Optical Music Recognition*) pipeline. The project uses cutting-edge computer vision to detect musical symbols and Digital Signal Processing (DSP) principles to convert spatial coordinates into discrete Hz frequencies.

Unlike traditional tools, PyClef focuses on **pure inference** and sub-millimeter precision, enabling the digitization of complex, handwritten, or printed sheet music.

### 🚀 Technologies
- **Deep Learning:** **YOLO11m** model fine-tuned for musical glyph detection.
- **Graphical Interface:** Developed in **PySide6 (Qt)** for a modern user experience.
- **DSP Engine:** Mathematical mapping of notes into MIDI files and synthesized audio.

### 🛠️ Roadmap & Status
- [x] **Neural Detection:** High-confidence symbol recognition via YOLO.
- [x] **GUI Development:** Intuitive interface with PySide6.
- [ ] **Real-time Synthesis:** (In progress) Immediate audio feedback of detected notes.
